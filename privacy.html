Política de Privacidade

Este site existe apenas para fornecer acesso à assistente LucIAnalista. Nenhum dado pessoal é coletado, armazenado ou compartilhado por este site.

1. Conversas com a assistente

As conversas acontecem diretamente na plataforma ChatGPT. Este site não tem acesso ao conteúdo das conversas e não armazena nenhuma informação digitada pelo usuário.

2. Cookies e navegação

Este site não utiliza cookies, rastreamento ou coleta de dados. Qualquer registro de navegação é exclusivamente da plataforma GitHub Pages.

3. Dados sensíveis

Não solicitamos, processamos ou armazenamos dados sensíveis de nenhum tipo.

4. Responsabilidade

LucIAnalista é uma presença simbólica para reflexão. Não substitui acompanhamento psicológico, psiquiátrico ou médico.

5. Contato

Para questões sobre este site, entre em contato com a criadora do projeto.
